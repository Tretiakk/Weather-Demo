# Weather
###### Portfolio Project

<img src="icon.svg" width="64" height="64">

**Project Overview:** A global weather application providing precise 24-hour and 7-day forecasts via OpenWeather API, automatically synchronized with the user's real-time location and fully localized for English, German, Spanish, and French.

**UI & Animations:** Developed with Jetpack Compose, featuring custom AnimatedVectorDrawable (AVD) for smooth, weather-specific visual transitions.

**Architecture & Testing:** Built on MVVM and SOLID principles with Hilt for DI, including unit testing via JUnit/Mockito and Proguard/R8 optimization.

**Tech Stack:** <br>
Kotlin, Jetpack Compose, MVVM, Coroutines, Flow, Hilt, Fused Location Provider API, Retrofit, OpenWeather API, JUnit, Mockito, AnimatedVectorDrawable, Gradle KTS.
